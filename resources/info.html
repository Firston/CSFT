Справка

===================

Логический значения

Для фильтров

1 - галочка поставлена, 0 - отсутствует

===================

===================

Идентификаторы полей

При работе с таблицами в заголовках проставлены типы заполнения полей. Их описание представлено ниже.

i - инкриментное поле, проставляется системой;
w - поля ручного ввода;
l - поля заполняется выбором из выпадающего списка;
c - поле проставления флага.

===================

===================

Проставлении общей коммиссии по терминалам

При проставлении коммиссии по умолчанию необходимо учитывать наличие записей о коммиссии по услуге. Если такие записи существуют, то возможно сделать только обновление уже существующий записей. К добавленному терминалу коммиссия проставляется в еденичном порядке.

===================

===================

Создание объекта типа "LIST"

При создании объекта типа "LIST" в поле "Код (w)" нужно проставлять точное наименование из таблицы "Справочники" колонка "Наименование справочника (w)".
